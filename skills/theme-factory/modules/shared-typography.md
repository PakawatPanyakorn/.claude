# Typography

**Never use:** Inter, Roboto, Arial, system-ui, Helvetica, or Space Grotesk.

All fonts must be loaded via Google Fonts `@import` in the HTML file.

## How to pick a pairing

1. Identify the theme's style keywords (e.g. "brutalist", "editorial", "cyberpunk").
2. **Scan every row in both tables** — collect all candidates whose `When` column contains any matching keyword. Do not stop at the first match.
3. From the candidate display fonts, choose the one whose `Why` description best fits the specific mood of this theme — not the first match, not the most obvious name. Prefer fonts lower in the list when top choices feel generic or overused.
4. Repeat step 3 for body fonts from the candidate body fonts.
5. The two chosen fonts form the pairing. Never pick both from the same table.
6. **Never default to the most famous font in a category** (e.g. Playfair Display for editorial, Orbitron for sci-fi). Treat all candidates as equally valid and pick based on the `Why` column's fit to the specific theme — not name recognition.
