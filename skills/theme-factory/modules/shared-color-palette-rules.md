# Color Palette Rules

| Rule | Detail |
| ---- | ------ |
| Contrast | ≥ 4.5:1 text/bg contrast |
| primaryLight | Same hue as `primary` |
| Dark theme bg | Lightness < 20% |
| Light theme bg | Lightness > 90% |
| Palette distribution | Use dominant hues with sharp accents — avoid timid even distributions |
| Anchor hues | Build cohesion around 1-2 anchor hues; accents must feel deliberately chosen |
| Pure black/white | **AVOID** `#000000` and `#ffffff` unless the theme's identity demands it |
